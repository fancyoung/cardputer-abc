# 数字键音频播放器

按数字键 `0`–`9` = 播放对应目录里的音频(歌、儿歌、自录的话都行)。一个数字对应一个目录,里面可放多首,默认顺序连播。

## 怎么加音频

把源文件丢进 `audio/<数字>/` 即可,支持 `mp3 / wav / m4a / aac / flac / ogg / opus`:

```
audio/
  1/  twinkle.mp3   abc-song.m4a        # 按 1 → 顺序播这俩
  2/  count-to-ten.mp3
  7/  bedtime.flac  cover.jpg           # cover.jpg = 这目录的兜底封面
```

然后打包(会转成固件要的 16-bit PCM mono 24kHz,并提取封面+标题):

```bash
python3 tools/build_pack.py            # 增量
```

产物在 `build/sd/packs/us-kid/<数字>/`,把整个 `build/sd/packs` 拷到 SD 卡根。

## 封面 + 标题从哪来

每首音频在播放器界面左边显示一个小封面方块 + 顶部标题:

- **封面**:优先用音频**内嵌封面图**(ID3 attached picture);没有则用该目录里的 `cover.jpg`(或目录里任意一张图)兜底;都没有 → 不显示封面,可视化占满整屏。
- **标题**:优先用音频的 **title 标签**(ID3);没有则用**文件名**(去扩展名)。

打包后每首会生成同名 `<stem>.wav` + `<stem>.jpg`(封面,有才生成)+ `<stem>.txt`(标题)。

## 设备上操作

- 数字键 `0`–`9` = 进对应目录,顺序连播。
- `◀` / `▶`(`,` / `/`)= 上一首 / 下一首(循环)。
- 播放界面 = 左封面 + 顶标题 + 右侧可视化(viz)。
- **可视化样式**在设置里切,或播放时按 **Fn+V** 快切:`number`(大数字)/ `bars`(声波条)/ `pulse`(圆脉冲)。设置和快捷键联动同一个值。
